# WaveLabs - Heating-System

## Prerequisites
- Install [paho-mqtt](https://pypi.org/project/paho-mqtt/#installation)
- `pip install numpy`
- Install [nicegui](https://nicegui.io/#installation)
- `pip install pytz`
